# GREP Command

## Example 1: -i

Here, we use the grep command with -i to look for escondido in the Vallarta-History text in the Berlitz 2 directory within travel guides. Adding -i searches through the text case insensitively. 

Source: [15 Practical Grep Command Examples In Linux / UNIX](https://www.thegeekstuff.com/2009/03/15-practical-unix-grep-command-examples/)

```
grep -i "escondido" written_2/travel_guides/berlitz2/Vallarta-History.txt
```
